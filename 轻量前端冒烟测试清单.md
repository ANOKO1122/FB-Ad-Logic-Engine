# 轻量前端冒烟测试清单

> 策略：不做大改，只做最小粒度验收
> 目标：确保前端基础功能正常，为后续完善做准备

---

## 📋 测试范围

### 最小粒度验收（不做大改）

#### 1. 登录功能 ✅
- [ ] 能否正常登录
- [ ] 登录后能否获取用户信息
- [ ] 登录状态是否持久化

**测试步骤**：
1. 打开 `http://localhost:3000/login`
2. 输入用户名和密码
3. 点击登录
4. 检查是否跳转到首页
5. 刷新页面，检查登录状态是否保持

---

#### 2. 账户切换 ✅
- [ ] 账户列表是否正常显示
- [ ] 能否切换账户
- [ ] 切换账户后数据是否更新

**测试步骤**：
1. 打开账户设置页面
2. 检查账户列表是否显示
3. 点击切换账户
4. 检查数据是否更新

---

#### 3. 基础列表渲染 ✅
- [ ] 规则列表是否正常渲染
- [ ] 数据监控列表是否正常渲染
- [ ] 列表数据是否正确显示

**测试步骤**：
1. 打开规则管理页面
2. 检查规则列表是否显示
3. 打开数据监控页面
4. 检查数据列表是否显示

---

#### 4. 基础操作 ✅
- [ ] 能否查看规则详情
- [ ] 能否查看数据详情
- [ ] 基础交互是否正常

**测试步骤**：
1. 点击规则卡片
2. 检查规则详情是否显示
3. 点击数据项
4. 检查数据详情是否显示

---
