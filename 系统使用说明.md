## 系统简介

**FB 智投系统** 是一个 Facebook 广告自动化管理平台，支持：
- 基于规则的自动化广告管理
- 多账户、多用户权限管理
- 广告数据监控与分析
- 规则执行日志审计

## 已完善功能详解

### 1. 用户认证系统

**功能说明：**
- 新用户可以注册账户
- 注册后需要等待管理员审核
- 审核通过后可以登录使用

**用户角色：**
| 角色 | 权限范围 |
|------|---------|
| admin | 所有权限，可查看所有账户数据，可审核用户 |
| 普通用户 | 只能查看自己负责的广告账户数据 |

---

### 2. 规则管理

**功能说明：**
- 创建自动化规则，设置触发条件和执行动作
- 支持多条件组合（AND/OR 逻辑）
- 支持指定目标广告范围

**规则配置项：**

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 规则名称 | 规则的标识名称 | "高花费低转化-暂停" |
| 广告账户 | 规则作用的账户 | act_123456789 |
| 目标层级 | 广告/广告组/广告系列 | 广告 (Ad) |
| 目标对象 | 指定的广告ID列表 | 可多选 |
| 触发条件 | 指标 + 运算符 + 阈值 | 花费 > 100 |
| 时间窗口 | 数据统计范围 | 今天/昨天/近7天 |
| 执行动作 | 满足条件时的操作 | 暂停广告 |
| 模拟模式 | 是否只记录不执行 | ✅ 推荐先开启 |

**支持的指标：**
| 指标 | 英文名 | 说明 |
|------|--------|------|
| 花费 | spend | 广告花费金额 |
| 展示次数 | impressions | 广告展示次数 |
| 点击次数 | clicks | 广告点击次数 |
| 购买次数 | purchases | 转化购买次数 |
| CPC | cpc | 单次点击成本 |
| CPM | cpm | 千次展示成本 |
| ROAS | roas | 广告支出回报率 |
| CPA | cpa | 单次获客成本 |

**支持的运算符：**
- `>` 大于
- `>=` 大于等于
- `<` 小于
- `<=` 小于等于
- `=` 等于
- `!=` 不等于

**支持的动作：**
| 动作 | 说明 | 状态 |
|------|------|------|
| 暂停广告 | 将广告状态改为暂停 | ✅ 已实现 |
| 启用广告 | 将广告状态改为启用 | ✅ 已实现 |
| 增加预算 | 增加广告组预算 | ⏳ 待开发 |
| 减少预算 | 减少广告组预算 | ⏳ 待开发 |

## 使用指南

### 快速开始

1. **登录系统**
   - 打开浏览器访问 `http://<服务器IP>:3000`
   - 输入账号密码登录

2. **创建第一条规则**
   - 进入「规则管理」页面
   - 点击「+ 新建规则」
   - 填写规则配置：
     - 规则名称：例如 "测试规则-花费大于100"
     - 选择广告账户
     - 选择目标广告（可多选）
     - 设置条件：花费 > 100
     - 选择动作：暂停广告
     - **⚠️ 勾选"模拟模式"**（重要！）
   - 点击「保存规则」

3. **测试规则**
   - 点击「立即运行所有规则」
   - 进入「执行日志」查看结果
   - 确认规则逻辑正确

### 注意事项

> ⚠️ **重要提醒**

1. **先用模拟模式测试**
   - 新创建的规则请先开启"模拟模式"
   - 确认规则逻辑正确后再关闭模拟模式

2. **目标对象要精确**
   - 规则只会作用于你选择的目标广告
   - 如果不选择目标对象，会作用于账户下所有广告（危险！）

3. **条件设置要合理**
   - 避免设置过于宽松的条件（如 花费 >= 0）
   - 这会导致大量广告被触发

4. **定期查看日志**
   - 关注执行日志中的"失败"记录
   - 及时发现和处理异常

## 常见问题

### Q: 为什么我创建的规则没有执行？
**A:** 请检查：
1. 规则是否已启用（开关是否打开）
2. 目标广告是否有数据（是否已同步）
3. 条件是否满足（广告数据是否达到阈值）

### Q: 为什么执行日志显示"跳过"？
**A:** "跳过"表示没有广告满足规则条件。这是正常的，说明规则正在工作，只是当前没有需要处理的广告。

### Q: 模拟模式和真实执行有什么区别？
**A:** 
- **模拟模式**：只记录日志，不实际改变广告状态
- **真实执行**：会调用 Facebook API 实际修改广告状态

### Q: 数据多久同步一次？
**A:** 
- Today 数据：每 10 分钟同步一次
- 近 7 天数据：每 30 分钟同步一次
- 也可以在「系统状态」页面手动触发同步

### Q: 为什么我看不到某些广告账户？
**A:** 
- 普通用户只能看到分配给自己的账户
- 如需查看更多账户，请联系管理员分配权限
