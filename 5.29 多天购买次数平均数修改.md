思路如下：

### 当前逻辑

```
loadCreatedSinceByAd → 查 structure_ads.created_time
    → createdDateByAd = { ad_1: '2026-05-08', ad_2: '2026-05-21' }
    → effectiveStart = createdDate + 1天
    → rangeStart = max(effectiveStart, 窗口起点)
```

### 改为

```
loadFirstSpendDateByAd（新增）
    → 查 daily_stats: SELECT ad_id, MIN(date) WHERE spend>0 GROUP BY ad_id
    → 补 ad_snapshots（兜底刚跑的新广告）
    → firstSpendDateByAd = { ad_1: '2026-05-10', ad_2: '2026-05-23' }
    → effectiveStart = firstSpendDate（不加 1 天）
    → rangeStart = max(effectiveStart, 窗口起点)
```
