# Drizzle ORM 快速开始指南

## 📋 第一步：创建数据库表

在执行 Drizzle 代码之前，需要先在 MySQL 中创建 `rules` 表。

### 方式 1：使用 SQL 脚本（推荐）

```bash
# 在项目根目录执行
mysql -u fb_ad -p fb_ad_brain < server/db/migrations/001_create_rules_table.sql
```

### 方式 2：手动执行 SQL

1. 打开 MySQL Workbench 或 Navicat
2. 连接到数据库 `fb_ad_brain`
3. 执行以下 SQL：

```sql
CREATE TABLE IF NOT EXISTS `rules` (
  `id` INT AUTO_INCREMENT PRIMARY KEY,
  `user_id` INT NOT NULL,
  `rule_name` VARCHAR(255) NOT NULL,
  `conditions` JSON NOT NULL,
  `actions` JSON NOT NULL,
  `enabled` BOOLEAN NOT NULL DEFAULT TRUE,
  `created_at` TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP,
  `updated_at` TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX `idx_user_id` (`user_id`),
  INDEX `idx_enabled` (`enabled`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 验证表是否创建成功

```sql
-- 查看表结构
DESCRIBE rules;

-- 查看表是否存在
SHOW TABLES LIKE 'rules';
```

## 📚 第三步：对比学习

### 推荐学习流程

1. **先执行 SQL**：在 MySQL 中手动执行 SQL 命令，理解结果
2. **再看 Drizzle**：查看对应的 Drizzle 代码
3. **运行代码**：执行 Drizzle 代码，验证结果是否一致
4. **修改尝试**：修改代码，看看会发生什么

### 示例：插入数据

**步骤 1：执行 SQL**
```sql
INSERT INTO rules (user_id, rule_name, conditions, actions, enabled)
VALUES (
  1,
  '测试规则',
  '[{"metric": "ctr", "operator": "lt", "value": 0.8}]',
  '[{"type": "pause_ad"}]',
  TRUE
);
```

**步骤 2：查看结果**
```sql
SELECT * FROM rules WHERE id = LAST_INSERT_ID();
```

**步骤 3：用 Drizzle 实现**
```javascript
import { db } from './drizzle.js'
import { rules } from './schema.js'

const newRule = await db.insert(rules).values({
  userId: 1,
  ruleName: '测试规则',
  conditions: [{ metric: 'ctr', operator: 'lt', value: 0.8 }],
  actions: [{ type: 'pause_ad' }],
  enabled: true
})

console.log('新规则 ID:', newRule[0].insertId)
```

**步骤 4：验证结果**
```sql
SELECT * FROM rules WHERE id = <刚才的ID>;
```

## 💡 常见问题

### Q1: 连接数据库失败？

**检查**：
1. `.env` 文件中的数据库配置是否正确
2. MySQL 服务是否启动
3. 数据库 `fb_ad_brain` 是否存在

### Q2: 表不存在？

**解决**：先执行 `001_create_rules_table.sql` 创建表

### Q3: Drizzle 代码报错？

**检查**：
1. 是否正确导入了 `db` 和 `rules`
2. 字段名是否正确（注意大小写）
3. 数据类型是否匹配

### Q4: 如何查看 Drizzle 生成的 SQL？

```javascript
import { sql } from 'drizzle-orm'

// 方式1：使用 toSQL()
const query = db.select().from(rules).where(eq(rules.userId, 1))
console.log(query.toSQL())

// 方式2：直接使用 sql 模板
const result = await db.execute(
  sql`SELECT * FROM rules WHERE user_id = ${1}`
)
```
