## 二、Thunder Client / curl 示例

### 2.1 GET /api/templates（普通用户可用）

```http
GET /api/templates
Cookie: token=<your_token>
```

预期：返回 `is_active=1` 的模板列表。

### 2.2 GET /api/admin/templates（需 admin）

```http
GET /api/admin/templates?include_inactive=1&limit=20&offset=0
Cookie: token=<admin_token>
```

### 2.3 POST /api/admin/templates（新建）

```http
POST /api/admin/templates
Content-Type: application/json
Cookie: token=<admin_token>

{
  "name": "测试模板",
  "slug": "test_template",
  "description": "用于验证",
  "sort_order": 99,
  "when_lines": [{"join": null, "metric": "spend", "operator": "gt", "value": 10}],
  "when_time_window": "today",
  "when_custom_range": null,
  "actions": [{"type": "pause_ad", "value": null}]
}
```

### 2.4 PUT /api/admin/templates/:id（更新，需 updated_at）

```http
PUT /api/admin/templates/4
Content-Type: application/json
Cookie: token=<admin_token>

{
  "name": "测试模板-已改",
  "description": "已更新",
  "sort_order": 99,
  "when_lines": [{"join": null, "metric": "spend", "operator": "gt", "value": 15}],
  "when_time_window": "today",
  "when_custom_range": null,
  "actions": [{"type": "pause_ad", "value": null}],
  "updated_at": "2025-02-12T08:00:00.000Z"
}
```

若 `updated_at` 与库中不一致 → 409 CONFLICT。

### 2.5 DELETE /api/admin/templates/:id（软删除）

```http
DELETE /api/admin/templates/4
Cookie: token=<admin_token>
```

### 2.6 POST /api/admin/templates/:id/restore（恢复）

```http
POST /api/admin/templates/4/restore
Cookie: token=<admin_token>
```

### 2.7 越权验证（普通用户调用 admin 接口）

```http
POST /api/admin/templates
Content-Type: application/json
Cookie: token=<non_admin_token>

{"name":"x","slug":"x","when_lines":[{"join":null,"metric":"spend","operator":"gt","value":0}],"when_time_window":"today","actions":[{"type":"pause_ad","value":null}]}
```

预期：403 Forbidden。

## 三、验收清单

- [ ] 管理员：新建/编辑/软删/恢复模板
- [ ] 列表分页：limit 默认 20、offset 正确
- [ ] 搜索：query 匹配 name/slug
- [ ] 乐观并发：PUT 时 updated_at 过旧 → 409
- [ ] 普通用户：看不到管理入口，规则页可加载模板
- [ ] 越权：非 admin 访问 admin 接口 → 403
