2. 规则名称：`阶段3验收-OR`；选好账户。
3. **触发条件**：
   - 时间窗口选 **今天**。
   - 第一行：**花费** **大于** **10**（首行无 AND/OR 正常）。
   - 点 **「+ 添加条件」**，**第二行左侧下拉选「OR」**，指标 **ROAS** **小于** **0.5**。
4. 执行动作、作用对象按需填，点 **保存规则**。
5. **SQL 确认**（可选）：

   SELECT id, rule_name, conditions FROM rules WHERE rule_name = '阶段3验收-OR' LIMIT 1;

   预期：`conditions.groups` 有 **2 个**（第一组 spend>10，第二组 roas<0.5）。
6. **回填验收**：在列表点该规则的 **编辑**。
