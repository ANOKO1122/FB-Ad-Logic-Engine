## 🔍 建议补充验证（可选）

### 归档数据一致性验证

**目的**：验证 `daily_stats` 中的数据是否与 `ad_snapshots` 中 `data_date = 昨天` 的数据一致

**验证 SQL**：
```sql
-- 对比 ad_snapshots 中昨天的数据（使用 data_date）
SELECT 
  account_id,
  data_date,
  COUNT(DISTINCT ad_id) as ad_count,
  SUM(spend) as total_spend
FROM ad_snapshots
WHERE data_date = DATE_SUB(CURDATE(), INTERVAL 1 DAY)
GROUP BY account_id, data_date
ORDER BY account_id
LIMIT 20;

-- 对比 daily_stats 中昨天的数据
SELECT 
  account_id,
  date,
  COUNT(DISTINCT ad_id) as ad_count,
  SUM(spend) as total_spend
FROM daily_stats
WHERE date = DATE_SUB(CURDATE(), INTERVAL 1 DAY)
GROUP BY account_id, date
ORDER BY account_id
LIMIT 20;
```

**预期结果**：
- 两个查询的结果应该基本一致（允许少量差异，因为归档是取"最后快照"）
- 如果差异较大，说明归档逻辑可能有问题
