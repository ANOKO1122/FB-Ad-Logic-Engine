## ⚠️ 核心问题：后端未接收 accountId

### 问题定位

**前端代码**（`src/views/RuleManager.vue`）：
```javascript
// 前端提交时传递了 accountId
accountId: selectedAccountId.value,  // 广告账户 ID
```

**后端路由**（`server/routes/rules.js`）：
```javascript
// ❌ 问题：后端没有接收 accountId 字段
const { 
  ruleName, 
  conditions, 
  actions, 
  enabled,
  targetLevel,
  targetIds,
  logicOperator,
  timezoneName,
  isSimulation
  // ❌ 缺少 accountId
} = req.body
```

**结果**：
- 前端传递了 `accountId`
- 但后端没有接收
- 导致规则创建时 `accountId` 为 `null`
- 反向索引优化无法生效

---
